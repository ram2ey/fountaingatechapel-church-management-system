# Fountain Gate Chapel

A comprehensive church management and community-building web application built for **Fountain Gate Chapel**.

## Features

- **📍 Multi-Branch Filtering**: Toggle feeds and directory information by specific local branch context.
- **🛡️ Leader Desk**: Direct administration dashboard for managing bulletins, group registrations, and service check-ins.
- **📈 Discipleship Path**: Track member commitments (Water Baptism, Volunteer Orientation, etc.) and class completions.
- **🏆 Interactive Youth Hub**: Safe space with points-based scripture memory challenges, discussions, and event lists.
- **🙏 Prayer Wall & Groups**: Communal fellowship pages supporting real-time likes, comments, and group-joining flows.
- **💳 Digital Giving**: Simulated payment flow for tithes, offerings, and charity funds.

## Local Development

### Prerequisites

- **Node.js** (v18+)

### Steps

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Configure Firebase**:
   Setup your Firebase client configs inside `firebase-applet-config.json` (or set up standard environment configurations).

3. **Start local Vite Development Server**:
   ```bash
   npm run dev
   ```
   Open [http://localhost:5173](http://localhost:5173) in your browser.

## Deployment to Vercel

The application is structured as a standard single-page React application. When deploying to Vercel, it uses the static build folder (`dist`) generated by:
```bash
npm run build
```

The custom `vercel.json` ensures that deep-linked paths are rewritten to `index.html` to avoid `404` errors on refresh.
